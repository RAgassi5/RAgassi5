## 📂 Projects

### 🔒 [Gmail Malicious Email Detector](https://github.com/RAgassi5/Gmail-Addon-malicious-email-detector)
**Python · FastAPI · React · Google Apps Script · AI · Security**

A layered Gmail security add-on that scores emails for maliciousness (0–100) and presents findings at multiple complexity levels — from quick verdicts for everyday users to detailed dashboards for security analysts.

**Highlights:**
- Multi-signal threat detection: phishing indicators, suspicious URLs, dangerous attachments, manipulative language
- Cross-references Google Safe Browsing API v4 and VirusTotal API v3
- Groq LLM in a bounded advisory role — never overrides hard evidence
- Privacy-preserving by design: only aggregate counts sent to backend, never email content
- Graceful degradation if external services are unavailable
- *Built during the Upwind Security Bootcamp*

### 🔍 [Wikipedia Search Engine](https://github.com/RAgassi5/InformationRetreival_SearchEngine.git)
**Python · Flask · Information Retrieval · Google Cloud Platform**

Full-scale search engine over the English Wikipedia corpus deployed as a public web service on GCP. Achieved **0.487 harmonic score** with zero failed queries and ~2.7s average latency.

**Highlights:**
- BM25-enhanced ranking combined with TF-IDF, title & anchor matching, PageRank, and page views
- Inverted indices stored in cloud storage, indexing **6M+ documents**
- Evaluated with Precision@5 and F1@30
- End-to-end cloud deployment
